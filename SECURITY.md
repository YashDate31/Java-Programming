# Security Policy

## Supported Versions

This repository is maintained as educational source code for Java practicals.
Latest `main` branch is supported.

## Reporting a Vulnerability

If you discover a security issue, please open a private report by contacting the maintainer through GitHub profile.

Please include:

- Affected file/path
- Reproduction steps
- Potential impact
- Suggested fix (if available)

We will review and respond as soon as possible.
